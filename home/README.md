# :house_with_garden: home

the home of my dotfiles. configured in, and managed by home-manager.

| name          | description                                                   |
| ------------- | ------------------------------------------------------------- |
| `profiles`    | defines each system profile                                   |
| `programs`    | houses home-manager configured programs                       |
| `services`    | houses services                                               |
| `wayland`     | houses wayland configuration, and configs for niri / hyprland |
| `default.nix` | defines home-manager config on all hosts                      |
